ABOUT THE SOFTWARE

Key features of our software

High end visualization

With the ability to scan a file and display its content on the main tabset, the software is fitted with the functionality to give you a full analysis of your file.If its a dataset, for example, it will show u the mean, median, quartile of each header in the file and much more.Not only that, but it also gives u the opportunity to present your data file in the form of scatter plots, histograms, bar graphs and our newly introduced 3D data visualization graphs such as 3D mesh plot, 3D scatter plot and much more.

Mechanism

The software lets you customize the appropriate file format setting.When it comes to plotting scatter plots, histograms or bar graphs, the software lets you pick which header variable goes to either the x-axis or the y-axis.And thats not all, our team has added additional features in the bar graph in that it allows you to download your graph in (png) form, zoom in and out of the graph to get even more detailed data, comparing data on hover and many more.

Design

As the software is associated with data analysis, it is coded and entirely in the R programming language using very prominent libraries in the R community such as Shiny for web development, ggplot2 and ploty for data visualization.The prototype software is public for use but the main software is private, thus only users who have paid for the software will recieve an invitation email from the same email used to pay for the software with a link to the software.

Support

Client satisfaction is our number one priority, therefore we try to make the software as easy to use as possible and on the other hand not negleting its functionality capabilities.We are currently working on moving all our data visualization techniques from ggplot2 to plotly as it has more functionalities for the end users.The software will also be updated and also conduct bug fixes regularly so as to make sure our clients get the value of their money.We are also open to reviews from you or any ideas or improvements you would want done on the software, just feel free to contact us.
